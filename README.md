# TIM GOON

Ini adalah projek akhir semester mata kuliah struktur data kelompok Tim Goon. Jenis projek yang dibuat yakni DSS (Decision Support System) mengenai pencarian katalog game berdasarkan genre dan harga.

Untuk penggunaan aplikasi ini hanya perlu melakukan instalasi pada raw source codenya, dan jangan lupa instalasi dan import library customtkinter agar GUI nya dapat berjalan. Disarankan untuk menjalankannya di Visual Studio Code. Lalu, jalankan programnya melalui terminal dengan command py (nama file).py , pastikan windows visual studio code kalian membuka folder yang berisi program yang ingin dijalani.
